## Overview

This is a production-ready MERN stack portfolio project designed with a strong focus on scalability, modular architecture, and performance optimization. 

The project follows a clean folder structure separating frontend and backend logic, ensuring maintainability and professional development standards.

## Project Structure

```text
Portfolio-main/
├── dist/                # Production build
├── server/              # Backend (Express API)
│   ├── .env             # Environment configuration
│   └── index.js         # Backend entry point
├── src/                 # Frontend source
│   ├── assets/          # Images & static files
│   ├── components/      # Reusable UI components
│   ├── constants/       # Static data
```

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Priyadarshandev/Portfolio-main.git
   cd Portfolio-main
   ```

2. **Install frontend dependencies:**
   ```bash
   npm install
   ```

3. **Install backend dependencies:**
   ```bash
   cd server
   npm install
   ```

4. **Environment Configuration:**
   Create a `.env` file inside the `server/` folder:
   ```env
   PORT=5000
   MONGO_URI=your_mongodb_connection_string
   JWT_SECRET=your_secret_key
   ```

5. **Run the application:**
   - **Frontend:** `npm run dev` (from root)
   - **Backend:** `node index.js` (from server folder)
