## Overview

This is a production-ready MERN stack portfolio project designed with a strong focus on scalability, modular architecture, and performance optimization. 

The project follows a clean folder structure separating frontend and backend logic, ensuring maintainability and professional development standards.

---

## Project Structure

```text
Portfolio-main/
├── client/              # Frontend (React + Vite)
│   ├── src/
│   │   ├── assets/      # Images & static files
│   │   ├── components/  # Reusable UI components
│   │   ├── pages/       # Page-level components (Home, etc.)
│   │   ├── hooks/       # Custom React hooks
│   │   ├── utils/       # Helper functions
│   │   ├── App.jsx      # Root component
│   │   └── main.jsx     # Application entry
│   ├── index.html
│   ├── package.json
│   └── vite.config.js
├── server/              # Backend (Node + Express)
│   ├── config/          # Database & service configs
│   ├── controllers/     # Business logic
│   ├── models/          # Mongoose models
│   ├── routes/          # Express routes
│   ├── middleware/      # Auth & error handling
│   ├── server.js        # Backend entry point
│   └── package.json
└── README.md
```

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Priyadarshandev/Portfolio-main.git
   cd Portfolio-main
   ```

2. **Setup Frontend:**
   ```bash
   cd client
   npm install
   npm run dev
   ```

3. **Setup Backend:**
   ```bash
   cd server
   npm install
   # Add your .env file
   npm run dev # or node server.js
   ```
---

## Production Build

To generate an optimized production build:
```bash
npm run build
```
The output will be generated inside the `dist/` folder.
